Setup Guide for the Warehouse Owner

  1. Create a new Google Sheet.
  2. Add these 9 column headers in the first row (Row 1):
    • No. (Column A)
    • Product Name (Column B)
    • Quantity (Column C)
    • Condition (Column D)
    • Catalogue Number (Column E)
    • Specs (Column F)
    • Image URL (Column G)
    • Mfg Date (Column H)
    • Arrival Date (Column I)
  3. Go to Extensions -> Apps Script.
  4. Delete any code in the editor, and paste the code from google_apps_script.js (found in this project's folder).
  5. Click the Save icon.
  6. Click the Deploy button (top right) -> Select New Deployment.
  7. Select type: Web App (click the gear icon to set type if needed).
  8. Change description to: "Inventory API".
  9. Set Execute as: "Me (your-email)".
  10. Set Who has access: "Anyone".
  11. Click Deploy. Authorize the script if prompted.
  12. Copy the Web app URL shown, paste it in the field above, and click Save & Connect!
