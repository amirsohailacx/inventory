# Feature List: ACX Instruments Inventory Manager

## Implemented
- [x] **8-Column Google Sheet Schema:** Read/write support for `No.`, `Product Name`, `Packing Size`, `Quantity`, `Condition`, `Customers`, `Catalogue Number`, and `Specs`.
- [x] **ACX Instruments Branding:** Replaced custom designs with your original logo image and matching corporate blue color scheme.
- [x] **Smart Unit Quantity Control:** Incrementing/decrementing parses number while preserving text suffixes (e.g. `47 Pcs` -> `48 Pcs`).
- [x] **Universal Table Sorting:** Click headers to sort by number, text, or quantity.
- [x] **Double-Click Inline Editing:** Double-click quantity cell to manually type edits.
- [x] **Comprehensive Global Search:** Instantly filters inventory by matching query against all 8 columns.
- [x] **Bulk Excel Import & Mapping:** Drop Excel files and dynamically map your Excel headers to the 8-column system database.
- [x] **Export to Spreadsheet:** Export current dashboard search view to a standard CSV sheet.
- [x] **Light/Dark Mode Theme:** Seamless toggle to switch color contrast, saving choices locally.
