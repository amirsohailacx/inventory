# Project Plan: Cloud-Shared Inventory Management Tool

## Objective
Build a cloud-shared, zero-cost, and user-friendly inventory management system for a small team with no coding experience. It uses Google Sheets as a database, Google Apps Script as a free API backend, and a modern single-page HTML5/CSS3/JS app as the frontend interface.

## Milestones
1. **Phase 1: Project Setup & Docs** (Created project plan, feature checklist, and bug tracker).
2. **Phase 2: Database Setup** (Define the Google Sheet columns and write the Google Apps Script code).
3. **Phase 3: Frontend Development** (Design a premium, mobile-responsive dashboard with search, filtering, and stock adjustments).
4. **Phase 4: Excel Import** (Allow the user to drop their Excel file to bulk-import items).
5. **Phase 5: User Setup Guide** (Write simple, step-by-step instructions for non-technical users).
